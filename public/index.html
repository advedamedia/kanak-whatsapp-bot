📱 WhatsApp Business API Setup Guide

Client ko ye steps khud karne hain. Unhe ye guide share karein.

1

Meta Business Account Banayein

Client browser mein jaaye: business.facebook.com → "Create Account" → Company name, email, phone enter karein.

2

Business Verification Complete Karein

Business Suite → Settings → Security Centre → "Start Verification". GST certificate, incorporation certificate, ya utility bill upload karein. (2-5 din lag sakte hain)

3

Meta Developer Account + App Banayein

developers.facebook.com → "My Apps" → "Create App" → "Business" type select karein → App banayein.

4

WhatsApp Product Add Karein

App Dashboard mein "Add Product" → "WhatsApp" → "Set Up" → Apna Business Account select karein.

5

Phone Number Register Karein + OTP Verify Karein

WhatsApp → Configuration → "Add Phone Number" → Number enter karein → OTP verify karein.

6

Access Token Generate Karein

WhatsApp → API Setup section mein → "Phone Number" note karein. System User Access Token banana: Business Settings → System Users → Token generate karein. Yeh token aapko next step mein dena hai.

7

Phone Number ID Copy Karein

WhatsApp → API Setup → "Phone Number ID" (numbers ka ek lambi string) copy karein. Yeh bhi aapko next step mein dena hai.

✅ Jab client ne Access Token aur Phone Number ID de diye, tab Next Step mein details fill karein.
